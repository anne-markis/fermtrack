# FERMTRACK CLI

### How to Run
Start the fermtrack server locally
```
# from ../fermtrack
% go mod vendor
% make start
```

Build and start the CLI
```
# from fermtrack dir
% go mod vendor
% cd cli
% go build; ./cli
```
